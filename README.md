# Automation Projects Analysis

## Project Overview
This project analyzes automation project data using Python and data visualization techniques.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset Information
The dataset contains 50,000 automation projects with details such as:
- Budget
- Annual Savings
- ROI Percentage
- Industry
- Department
- Country
- AI Adoption
- Project Status

## Analysis Performed

### Pandas
- Dataset Overview
- Missing Value Analysis
- Project Status Analysis
- Country-wise Project Distribution
- Industry Performance
  

### NumPy
- Budget Statistics
- Employee Hours Saved Analysis
- ROI Categorization

### Matplotlib
- Project Status Bar Chart
- Budget vs Savings Scatter Plot
- Industry-wise Average ROI
- Country-wise Project Distribution

### Seaborn
- Correlation Heatmap
- Budget Distribution
- ROI Distribution
- Boxplots and Countplots

## Key Insights
- Completed projects form the majority of projects.
- AI-enabled projects account for over 55% of all projects.
- Higher-budget projects tend to generate higher annual savings.
- Most projects achieve high ROI (>100%).
